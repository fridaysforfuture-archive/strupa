| StruPa_v2.1 | 01.04.2021       | -        | aktuell legitimiertes StruPa | Finanzkonzept_v3.0 (25.02.2021)               |

**Dokumente zum Bearbeiten**

**Abstimmung/Änderunganträge:**

- 📌 Auflistung aller eingearbeiteten Anträge:
- 📌 Abstimmung über einen Strukturänderungsantrag zur zukünftigen Zuständigkeit für das Aktualisieren des Strukturpapiers  
  - ℹ️ Die Struktur AG hat beschlossen sich aufzulösen. Sie hat einen letzten Strukturänderungsantrag formuliert, in dem die Zuständigkeit für das Aktualisieren des Strukturpapiers an die CTF gegeben werden soll. Über diesen wurde nun abgestimmt. (01.03.2021 bis 11.03.2021)  
  - **Teilgenommen:** 25 OGs  
  - **Ja:** 87,50%  
  - **Nein:** 12,50%  
  - **Enthaltungen:** eine  
  - ▶️ Somit ist dieser Änderungsantrag angenommen und die CTF wird sich in Zukunft um das Aktualisieren des Strukturpapieres kümmern.  
  - 👉 [Änderungsantrag](https://fffutu.re/strukturpapier)  
  - 👉 [Liebesbrief an die Bewegung](https://fffutu.re/liebesbrief)  
  - 👉 [Genaue Aufschlüsselung](https://fffutu.re/ljmAfX)

- 📌 Abstimmung über einen Strukturänderungsantrag zu Sanktionen bei übergriffigen Verhalten  
  - [Triggerwarnung: Thema übergriffiges Verhalten gegenüber PoC und FINTA*, Täter:Innenschutz]  
  - ℹ️ Einzelpersonen haben einen Änderungsantrag ausgearbeitet, der Betroffene von übergriffigen Verhalten durch beispielsweise einen Ausschluss der übergriffigen Person schützen soll. (21.03.2021 bis 01.04.2021)  
  - **Teilgenommen:** 48 OGs  
  - **Ja:** 87,23%  
  - **Nein:** 12,77%  
  - **Enthaltungen:** eine  
  - ▶️ Dieser Antrag wurde also angenommen.  
  - 👉 [Änderungsantrag & Erklärung](https://fffutu.re/CQpySD)  
  - 👉 [Genaue Aufschlüsselung](https://fffutu.re/EAOny0)
